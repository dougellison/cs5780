How does it compare to BackboneJS


Below are excerpts directly from the BackboneJS website http://backbonejs.org/#faq

  • View agnostic
  • Goal is to be a library
  • Avoids Two way data-binding
  • The focus is on supplying you with helpful methods to manipulate and query your data, not on HTML widgets or reinventing the JavaScript object model.
  • HTML and Logic are completely separated
